本人上海地区电信网络
就单纯访问速度而已，百度的最快，其次是阿里公共DNS

本人推荐几组DNS配置：

配置一：
- 首选：180.76.76.76 
- 备用：9.9.9.9 或 8.8.8.8

配置二：
- 首选：223.5.5.5
- 备用：8.8.8.8 或 9.9.9.9

配置三：
- 首选：180.76.76.76 
- 备用：208.67.222.222 或 208.67.220.220

配置四：
- 首选：223.5.5.5
- 备用：208.67.222.222 或 208.67.220.220

这样上国内网站时，百度保证了解析速度，
碰到无法解析的国外网址时，IBM的DNS还能助我们一臂之力。

由于地区性差异，我的配置不一定适合所有人，可以使用

```bash

ping dnsIp -t

```

进行观察响应或丢包情况。

也可通过 

```bash

tracert dnsIp

```

进行网络追踪观察路由情况。
